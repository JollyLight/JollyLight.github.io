常用快捷键小记

2020-11-06- 2020-11-24
备忘笔记-快捷键

这篇文章可能对你没什么帮助，_(:з)∠)_主要是记给自己看的，只记录了我个人觉得应该常用但我不常用的快捷键(*/ω＼*)，没有列出我很熟悉的与很不熟悉的快捷键(如果你真那么巧恰好和我熟悉的和不熟悉的快捷键一样那就应该对你也有很大帮助)

微软输入法（拼音）

，：上翻页

。：下翻页

↓ ：下一个字

↑ ：上一个字

Ctrl + Shift + B ：表情（包含各种符号）

Ctrl + ← / ↑ / →/ ↓ ：切换表情（包含各种符号）

Edge浏览器

Ctrl + Tab：切换到下一个标签页

Ctrl + Shift + Tab：切换到上一个标签页

Ctrl + T：新建标签页

Ctrl + Shift + T：恢复上一个关闭的标签页

Alt + ←/→：对同一标签页的返回与前进操作

Ctrl + E：在当前标签打开搜索引擎查询

Ctrl + H：历史记录

Ctrl + W：关闭当前标签页

Ctrl + F5：强制刷新当前页面（部署了博客后页面没更新可以试试这个）

空格键：向下滚动网页（浏览学校的pdf非常好用）

Shift+ 空格键：向上滚动网页

Windows10

win + V：打开剪贴板（可以省去一些情况下需要2个页面来回跳转复制粘贴的麻烦)

win + Shift + S：截图（自动保存至剪贴板，但不能自动保存至某个文件夹内）（发现了更好的方法，如下）

（以下截图的前提是将设置→轻松使用→键盘→屏幕截图快捷方式中的使用PrtScn按键打开截图打开）

PrtSc：截图并保存至剪贴板

win + PrtSc：截图保存至剪贴板并保存至文件夹（默认保存在了C:\Users\你的用户名\Pictures\屏幕截图文件夹下，你可以右击该文件夹选择属性→位置→移动…来选择你想保存在的文件夹，这是我找到的最舒服的截图方式了[](￣▽￣)*）

Alt + F4：关闭当前页面（用Fn键控制时按键的顺序是Fn + Alt + F4）
